Sim, temos uma alternativa linux para a Time Machine do macosx, se trata do Timeshift, o estou utilizando no meu note e já me salvou 2 vezes. Para que ele serve ?

O Timeshift utiliza do RSync (rsync é um utilitário amplamente usado para manter cópias de um arquivo em dois sistemas de computadores ao mesmo tempo - wikipédia), e faz uma cópia do seu sistema operacional e te dá a possibilidade de poder retornar a uma versão especifica que foi realizada o backup. Muito útil para quem gosta de mexer no sistema operacional e se esquece do que fez.

Não foi me alongar muito aqui, já existe muito material na internet sobre ele. Queria adicionar somente uma dica, faça uma partição separada do sistema operacional ou utilize um HD-externo para fazer o backup.
